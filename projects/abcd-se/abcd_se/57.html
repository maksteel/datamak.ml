4.3.1
Baseline Experiments
The purpose of these experiments were to make sure that using GNU Octave instead of
Matlab
2
produces similar results. In baseline experiments, we performed model search and
prediction without any change in ABCD system i.e. we used the Algorithm
2
approach to
discover covariance kernel function greedily.
We did not experiment with any of the system parameters with different values, as
the purpose was to establish a baseline, we kept same values as set by original thesis
[
14
], for example, we used the same random seeds, kept
iters
= 250,
full
iters
= 10,
subset
size
= 250 and
desired
depth
= 10. We performed these experiments on four
out of 13 univariate datasets used in original thesis [
14
], they were 01-airline, 06-internet,
07-call-centre and 08-radio.
2
Matlab was used by Lloyd et al. [
15
] for their experiments
