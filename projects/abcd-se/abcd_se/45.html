Discovering composite kernel

has the potential to greatly change a Gaussian process model, without requiring to change
inference or learning procedures. We have also discussed that using a small number of
base kernels compositionally i.e. by adding or multiplying we can construct a variety of
sophisticated structures.

system performs three steps as formalised in Algorithm

variables, like base kernels, operators, criterion used for model evaluation are all initialised.
The kernel search is performed next, this step discovers the composite kernel using Gaussian
process regression that best describes the given training dataset. Algorithm

the steps involved in kernel search. Finally, ABCD system performs prediction on test
data to verify how well the discovered model generalises on unseen data.
