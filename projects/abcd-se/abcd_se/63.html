5.1
Baseline experiments
ABCD system runs parallel jobs to search for best covariance kernel composition. Gaussian
process regression is performed using GPML toolbox. Lloyd [
14
] used Matlab to interface
with GPML toolbox. We decided to use GNU Octave for our experiments, due to reason
that Matlab (statistics package) licenses on NeSI were shared. The license sharing resulted
in long waiting times and ABCD system became inefficient in running parallel model
search jobs.
For that reason, we performed these baseline experiments to assess the impact of using
GNU Octave software for interfacing with GPML toolbox inside ABCD system.
