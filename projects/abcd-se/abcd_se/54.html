4.2
Proposed methods
In this section, we introduce the methods that we developed to test the alternative
hypotheses. We would first describe the model search with top-k strategy while contrasting
it with greedy search approach. We then move on to Cross Validated Subset of Data (CV
SoD) a novel model approximation technique to Gaussian process regression.
4.2.1
Kernel search with top-k strategy
We, now, discuss how covariance function or composite kernel is discovered using top-three
search method. Like greedy approach, top-three method starts by proposing models with
only base kernels. These proposed models are evaluated, scored and ranked. Top-three
ranked models are selected for next level of search.
